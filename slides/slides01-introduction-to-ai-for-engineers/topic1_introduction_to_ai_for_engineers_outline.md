# Introduction to Machine Learning
- ML = algorithms that learn patterns
- Requires historical data
- Produces predictions or decisions

# Types of Machine Learning
- Supervised learning
    - Regression
    - Classification
- Unsupervised learning
    - Clustering
    - Anomaly detection

# When to Use ML in Engineering
- Patterns too complex for equations
- Many interacting variables
- Predictions needed

# When Not to Use ML
- No data
- Simple systems with known formulas
- Safety-critical without explainability
