# Datasets (Not Included)

To avoid licensing and size issues, raw datasets are **not** stored in this repository.

Recommended open datasets:
- Open Power System Data (OPSD) for load and energy forecasting.
- UCI Condition Monitoring of Hydraulic Systems dataset.
- Case Western Reserve University Bearing Data Center vibration dataset.
- Public crack/surface defect image datasets for computer vision labs.

Download the datasets manually and place them into appropriate `labs/` subfolders.
